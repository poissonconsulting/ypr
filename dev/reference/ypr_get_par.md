# Get Parameter Value

Get Parameter Value

## Usage

``` r
ypr_get_par(object, par = "pi")
```

## Arguments

- object:

  A ypr object.

- par:

  A string of the parameter.

## Value

A numeric or integer scalar or vector of the parameter value.

## Examples

``` r
ypr_get_par(ypr_population())
#> [1] 0.2
```
