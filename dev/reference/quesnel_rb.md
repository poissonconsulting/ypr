# Quesnel Lake Rainbow Trout Population Parameters

The population parameters for Rainbow Trout in Quesnel Lake, BC.

## Usage

``` r
quesnel_rb
```

## Format

An object of class
[`ypr_population()`](https://poissonconsulting.github.io/ypr/dev/reference/ypr_population.md).

## Details

The estimates are liable to change and should not be used for
management.
