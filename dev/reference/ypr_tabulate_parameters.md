# Tabulate Population Parameters

Tabulate Population Parameters

## Usage

``` r
ypr_tabulate_parameters(population)
```

## Arguments

- population:

  An object of class
  [`ypr_population()`](https://poissonconsulting.github.io/ypr/dev/reference/ypr_population.md).

## Value

A table of population parameters
